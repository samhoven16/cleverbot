# CleverBot — AI Assistant for Discord

CleverBot is a powerful AI assistant for Discord servers, powered by Claude AI.

## Commands
- `!ask [question]` — Ask the AI anything
- `!status` — Check your plan and usage
- `!upgrade` — View premium options

## Pricing
